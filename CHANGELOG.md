# Changelog

All notable changes to **NxthPress** will be documented in this file.  
This project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] — 2025-09-24  

### Initial Release 🎉

- **Page Builder** with modular blocks and live SEO preview  
- **Blog engine** with tags, pagination, Markdown, and per-post SEO  
- **Next.js 15 App Router** with React Server Components and ISR  
- **Prisma + PostgreSQL** with migrations and seed data  
- **Auth.js v5 (NextAuth)** integration for authentication and admin  
- **Cloudinary uploads** for image management  
- **Docker Compose** for local PostgreSQL setup
